<Section icon={Shield} title="Systemanforderungen und Performance">

Um ein reibungsloses Spielerlebnis zu gewährleisten, hat das Entwicklungsteam die Game-Engine für moderne Hardware optimiert. PC-Spieler müssen sicherstellen, dass ihre Systeme die Mindestanforderungen erfüllen, um das Spiel mit stabilen Bildraten auszuführen. Das Spiel nutzt fortschrittliche Beleuchtungs- und Partikeleffekte, die erheblich von Solid-State-Speichern (SSD) profitieren.

| Komponente | Mindestanforderungen | Empfohlene Anforderungen |
| :--- | :--- | :--- |
| Betriebssystem | Windows 10 (64-Bit) | Windows 11 (64-Bit) |
| Prozessor | Intel Core i5-8400 | Intel Core i7-10700K |
| Arbeitsspeicher | 8 GB RAM | 16 GB RAM |
| Grafikkarte | NVIDIA GTX 1060 | NVIDIA RTX 3060 |
| Speicherplatz | 50 GB SSD | 50 GB SSD |

Konsolenversionen werden über dedizierte Performance- und Qualitätsmodi verfügen. Der Performance-Modus zielt auf stabile 60 Bilder pro Sekunde bei dynamischer Auflösung ab, während der Qualitätsmodus eine native 4K-Auflösung mit verbesserten Grafikfunktionen wie Raytracing-Reflektionen und verbesserter Schattenqualität priorisiert.
