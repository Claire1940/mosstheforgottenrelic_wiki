<Section icon={Map} title="「Moss: 忘れられた秘宝」泥の寺院への進入">

**「Moss: 忘れられた秘宝」の泥の寺院**は、Book Iにおける最初の大きな難所の1つです。冒険のチャプター3として、基本的な森林の探索から、クイルとリーダーの深い絆を必要とする複雑な建築パズルへと移行します。泥の寺院に入ると、プレイヤーは水没した遺跡や、何世紀もの間眠っていた古代の仕掛けを突破していくことになります。 

このセクションは、ゲームの残りの部分で使われる「パズルの共通言語」を確立する重要なエリアです。特定の重量にのみ反応する感圧版、固定し続けなければならない回転する橋、そして拡張コンテンツ「黄昏の庭園」の導入部分に遭遇します。雰囲気は明るい「清らかな森」から、より厳かでエメラルド色に彩られた地下環境へと変化し、2026年リリースの「忘れられた秘宝」のテーマを際立たせています。

| **パズル** | 回転する橋を揃える | 上層の聖域へのアクセス |
| **秘密** | 黄昏のポータルを見つける | 「黄昏の庭園」DLCへの進入 |

<Section icon={Target} title="中心的なパズル要素とリーダーのインタラクション">

**「Moss: 忘れられた秘宝」の泥の寺院**におけるパズルは、クイルがアスレチックをこなす一方で、リーダーが物理世界を操作する能力に大きく依存しています。これまでのチャプターとは異なり、泥の寺院では仕掛けを「保持」し続ける必要があります。リーダーが回転する足場から手を離すと、足場は初期状態に戻ってしまうことが多く、クイルが閉じ込められたり、下の泥水に落ちたりする危険があります。

2026年の非VR版では、これらの操作が標準的なコントローラーやマウス＆キーボードの設定に適応されています。リーダーのカーソルを使って、リーダーのシンボルが描かれた青く光るオブジェクトを掴み、回転、引っ張り、または持ち上げる必要があります。この同時並行のタスク処理こそが、泥の寺院の体験の核心です。

<CardGrid cols={3}>
  <Card title="回転する橋">
    - **操作**: 円形の歯車アイコンを掴む。
    - **機能**: 足場を90度または180度回転させる。
    - **戦略**: まずクイルを橋の上に移動させ、その後に回転させる。
  </Card>
  <Card title="感圧版">
    - **操作**: 重さが必要（クイルまたは敵）。
    - **機能**: 門を開く、または遠くの崖を上昇させる。
    - **戦略**: リーダーを使ってアーケインの敵をプレートの上にドラッグする。
  </Card>
  <Card title="移動可能なブロック">
    - **操作**: ハイライトされた石のキューブをドラッグする。
    - **機能**: 登るための高さを提供する。
    - **戦略**: クイルのために、傷のある壁の近くにブロックを配置する。
  </Card>
</CardGrid>

| 仕掛け | リーダーのアクション | クイルのアクション |
| :--- | :--- | :--- |
| **エレベーター昇降機** | レバーを上に保持する | 足場の上に立つ |
| **ゲートロック** | 光る鎖を引く | 開口部をダッシュで通り抜ける |
| **敵のスタン** | アーケインの甲虫を掴む | 剣のフィニッシャーを実行する |

<Callout type="warning" title="時間制限のあるシーケンス">
泥の寺院の一部の足場には**タイマー**が設定されています。リーダーが操作した後、3〜5秒でリセットされることがあります。仕掛けを作動させる前に、クイルが開始地点の崖のギリギリに位置していることを確認してください。
</Callout>

<Section icon={Sparkles} title="泥の寺院ステップバイステップ攻略">

**「Moss: 忘れられた秘宝」の泥の寺院**を無事にクリアするには、以下の手順に従ってください。このウォークスルーでは、チャプター3の入り口から最終の部屋までのルートをカバーしており、重要な進行ポイントを見逃さないようにします。

<Steps>
  <Step num="1" title="入り口の門">
    寺院の外に到着すると、大きな施錠された門が見えます。左側にある動かせる石のブロックを探してください。リーダーとして、ブロックを壁の右側にドラッグします。クイルに切り替え、ブロックを登って上部にあるレバーに到達します。このレバーを引くと、メインの入り口が開きます。
  </Step>
  <Step num="2" title="最初の泥の部屋">
    床が水没しています。クイルを最初の固定された足場に移動させます。リーダーを使って中央の橋をクイルの方向に回転させます。橋を渡り、クイルがその上に立っている間にもう一度橋を回転させて、奥の出口に到達します。
  </Step>
  <Step num="3" title="アーケインの操作">
    出口を開く感圧版がありますが、クイルが降りるとリセットされてしまいます。部屋の中にいる甲虫のようなアーケインの敵を見つけてください。リーダーを使ってその敵を掴んで固定し、プレートの上に直接ドラッグします。これにより扉が開いたままになり、クイルが通過できるようになります。
  </Step>
  <Step num="4" title="黄昏のポータル">
    中央のハブを離れる前に、大きな門の脇を確認してください。紫色のきらめくエフェクトが見えるはずです。これが最初の**「黄昏の庭園」ポータル**です。ここに入るのは任意ですが、後のBook IIのパズルで役立つアップグレードされた装備を入手できるため、入ることをお勧めします。
  </Step>
  <Step num="5" title="垂直上昇">
    最後の部屋では、3つのリフトを順番に上昇させる必要があります。クイルを最初のリフトに移動させ、リーダーでそれを上昇させ、2番目のリフトにジャンプします。3番目のリフトでもこれを繰り返しますが、バルコニーから撃ってくる遠距離攻撃の敵「ティッカー」に注意してください。
  </Step>
</Steps>

<Callout type="success" title="チェックポイント到達">
ゲームは**定期的なオートセーブシステム**を使用しています。垂直上昇中にクイルが落下した場合、寺院の最初からではなく、その特定の部屋の最初から再スタートとなります。
</Callout>

</Section>

<Section icon={Swords} title="チャプター3の戦闘戦略">

**「Moss: 忘れられた秘宝」の泥の寺院**での戦闘では、リーダーによる支援を伴うクラウドコントロール（敵の制御）が必要になります。近接攻撃を行う「スクリーチャー」（甲虫）や、遠距離から爆発物を放つ「ティッカー」に遭遇します。寺院の狭い通路では、囲まれることが最大の脅威となります。

パズルに集中したいプレイヤーのために「戦闘スキップ」機能も用意されていますが、戦闘に挑む場合はクイルの剣とリーダーの掴みアクションの相乗効果が不可欠です。リーダーがスクリーチャーを掴むことで攻撃を防ぎ、クイルが他の脅威に集中したり、拘束された標的に大ダメージを与えたりすることができます。

| 敵の種類 | 主な攻撃 | 弱点 | 戦略 |
| :--- | :--- | :--- | :--- |
| **スクリーチャー** | 近接ひっかき | 旋回速度が遅い | 背後に回避して攻撃する |
| **ティッカー** | 遠距離発射体 | 低体力 | リーダーを使って弾道をそらす |
| **フォージド・ドローン** | 連射 | 光るコア | ドローンがオーバーヒートした後に攻撃する |

| 戦闘アビリティ | アンロック状態 | 戦闘での実用性 |
| :--- | :--- | :--- |
| **ソードコンボ** | 初期状態 | 甲虫に対する基本ダメージ |
| **回避ロール** | 初期状態 | ティッカーの爆風を避けるために不可欠 |
| **リーダーヒール** | 初期状態 | クイルを掴んで体力を回復する |

<Callout type="info" title="クイルの回復">
クイルの体力が低下すると（画面が灰色がかってきます）、リーダーを使って**手を伸ばし、彼女に触れてください**。インタラクションボタンを長押しすると緑色のエネルギーが脈動し、彼女の体力を全回復させます。敵から離れた安全な隅で行いましょう。
</Callout>

</Section>
