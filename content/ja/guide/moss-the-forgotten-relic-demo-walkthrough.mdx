<Section icon={Swords} title="戦闘の基本とナビゲーションの戦術">

**「Moss: 忘れられた秘宝」**の戦闘はスピーディーですが、戦術的な意識が必要です。クイルは敏捷ですが、彼女のすべてのアビリティを使いこなさなければ、アーケインの構築物（Arcane constructs）に簡単に圧倒されてしまいます。体験版では、基本的な剣術、回避、そして敵を掴んでクイルのトドメを支援する重要な「読者」のインタラクションが紹介されます。

<CardGrid cols={3}>
  <Card title="剣術">
    - **素早い攻撃**: 敵のアーマーを破壊するための素早い一撃。
    - **草刈り**: 隠されたシャードを見つけるのに便利。
    - **コンボの可能性**: 攻撃を繋げてより高いダメージを与える。
  </Card>
  <Card title="回避行動">
    - **回避ロール**: アーケインの飛び道具を避けるために不可欠。
    - **ジャンプ攻撃**: 飛行するセンチネル（見張り）を攻撃するのに有効。
    - **ポジショニング**: 追い詰められないよう常に動き続ける。
  </Card>
  <Card title="「読者」の支援">
    - **マインドコントロール**: 敵を掴んでスタン（気絶）させる。
    - **回復**: 集めたエネルギーを使ってクイルの体力を回復する。
    - **環境を利用したキル**: 敵を罠や危険地帯に投げ落とす。
  </Card>
</CardGrid>

環境を探索する際は、「読者」がインタラクトできるオブジェクトを見逃さない鋭い目が必要です。これらは通常、かすかな光や質感の違いで強調されています。例えば、足場を前後に動かすことで、普通では届かない高所にあるスクロールにクイルが到達できるようになります。

| 敵のタイプ | 主な攻撃方法 | 対処法 |
| :--- | :--- | :--- |
| **アーケイン・ビートル** | 近接突進 | 回避してから反撃 |
| **フライング・センチネル** | 遠距離弾 | 弾き返すかジャンプ攻撃 |
| **コンストラクト・ガード** | 叩きつけ（重攻撃） | マインドコントロールでスタンさせてから攻撃 |

<Callout type="warning" title="戦闘の警告">
むやみに剣を振り回しすぎないように注意しましょう。クイルの体力は限られており、複数の敵に囲まれるとあっという間にゲームオーバーになってしまいます。「読者」の掴み能力を使って、敵の数をコントロールしてください。
</Callout>

<Section icon={Target} title="温室：キャリルのハンマーの入手">

温室を進むにつれて、パズルはより複雑になります。このエリアでは、クイルに絶大な力を与える一方で移動速度を低下させる伝説の遺物、**キャリルのハンマー（Hammer of Caryll）**が登場します。このセクションは、2つの武器を切り替えるシステムを理解する上で非常に重要です。

<Steps>
  <Step num="1" title="ハンマーの祭壇を見つける">
    生い茂る植物をかき分けて進み、中央の台座に到達します。キャリルのハンマーは、重量をテーマにしたパズルの奥に封印されています。
  </Step>
  <Step num="2" title="ハンマーの仕掛けを起動する">
    「読者」の力を使って、部屋の中にあるハンマーのシンボルを操作します。遺物をチャージするには、特定の順序でスイッチを押す必要があります。
  </Step>
  <Step num="3" title="チャージ攻撃をマスターする">
    ハンマーを入手したら、攻撃ボタンを長押ししてチャージします。この一撃は、剣では壊せない頑丈な障壁を破壊したり、床のスイッチを起動したりできます。
  </Step>
  <Step num="4" title="隙間を渡る">
    ハンマーの重さを利用して、通常ならクイルがバランスを崩して滑り落ちてしまうような高速で動く足場の上に留まります。
  </Step>
</Steps>

| パズルの仕掛け | インタラクション方法 | 結果 |
| :--- | :--- | :--- |
| **ハンマーのシンボル** | チャージしたハンマー攻撃 | 機械仕掛けの扉を開く |
| **温室の門** | 読者が引く ＋ クイルのジャンプ | 番人への道が開かれる |
| **加重プレート** | クイル（ハンマー装備時） | 隠された昇降機を起動する |

<Callout type="success" title="新武器アンロック">
キャリルのハンマーは戦闘のためだけの道具ではありません。体験版の各所にある光る青いクリスタルを破壊して、隠された通路や追加のスクロールを見つけましょう。
</Callout>

</Section>

<Section icon={Shield} title="最終決戦：シンダーナイトの番人 攻略法">

体験版のクライマックスは、**シンダーナイトの番人（Cinder Knight Warden）**との対決です。この巨大なボスは、アーガス叔父さんが言及していた「野蛮で戦争に執着する」守護者を象徴しています。この戦闘は、剣で直接戦うよりも、キャリルのハンマーをいかに効果的に使うかが鍵となります。

**ボスのフェーズと戦術:**
1. **ハンドスラム**: 番人は巨大なガントレットでクイルを押しつぶそうとします。最初の衝撃を回避しましょう。
2. **隙だらけの状態**: スラム攻撃の後、番人の手は数秒間地面に留まります。
3. **ハンマーチャージ**: キャリルのハンマーに切り替え、番人の拳（指関節）にフルチャージした一撃を叩き込みます。
4. **環境とのインタラクション**: 番人が咆哮を上げたら、「読者」の力で天井から瓦礫を引きずり下ろし、より長い時間スタンさせます。

| ボスの攻撃 | 予兆 | 回避方法 |
| :--- | :--- | :--- |
| **オーバーヘッドスラム** | 番人が手を高く上げる | すぐに横へローリング回避 |
| **シンダースイープ** | 床に火が現れる | 衝撃波をジャンプで飛び越える |
| **アーケインロア** | 画面が激しく揺れる | 石柱の裏に隠れる |

<Callout type="warning" title="ボス戦の重要なヒント">
剣で番人の体を直接攻撃しようとしないでください。ダメージは一切通りません。手のスラム攻撃を待ち、キャリルのハンマーを使って効果的なダメージを与える必要があります。
</Callout>
