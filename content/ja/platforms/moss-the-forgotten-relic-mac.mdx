<Section icon={Shield} title="互換性の概要">

macOSで最新のゲームを実行するには、特に発売直後にネイティブ版が提供されていない場合、互換レイヤーを利用する必要があります。本作は、高度なグラフィック環境と複雑な物理演算を特徴としており、堅牢なハードウェアパフォーマンスが求められます。コンソール版ではエンジンがネイティブに最適化されていますが、MacユーザーもApple Silicon向けに構築された最新の互換ユーティリティを活用することで、優れた動作結果を得ることができます。

Apple Siliconチップは、CPUとGPUがリソースを動的に共有できるユニファイドメモリ構造を採用しています。このアーキテクチャは、コンソール級のグラフィックスとMacのハードウェアの間のギャップを埋めるのに役立ち、Windows向けアプリケーションを実行する際にスムーズな変換プロセスを提供します。ただし、互換性は互換レイヤーソフトの設定や、デバイスにインストールされているmacOSのバージョンに大きく依存します。

<Section icon={Map} title="macOSでのセットアップ方法">

Macでゲームを正常に実行するには、技術的な知識やハードウェア構成に合わせて互換レイヤーを選択する必要があります。ツールによって、最適化のレベル、使いやすさ、互換性が異なります。

Whiskyのようなラッパーアプリケーションを使用すると、AppleのGame Porting Toolkitのインストールが簡略化され、コマンドラインを使用せずに環境を構築できます。以下の手順に従って環境を設定してください。

<Steps>
  <Step num="1" title="WhiskyまたはCrossOverのインストール">
    お好みの互換レイヤーをダウンロードしてインストールします。AppleのGame Porting Toolkitをベースにした、無料で使いやすいインターフェースを提供するWhiskyがおすすめです。
  </Step>
  <Step num="2" title="Windowsボトルの作成">
    アプリケーション内で新しいWindows 10またはWindows 11の「ボトル（Bottle）」を作成します。最適なパフォーマンスを得るために、ボトル設定でDXVKとMSyncまたはESyncが有効になっていることを確認してください。
  </Step>
  <Step num="3" title="ボトル内へのSteamのインストール">
    Windows用のSteamインストーラー（.exe）をダウンロードし、互換レイヤー経由で実行します。アカウントにログインし、ゲームクライアントを直接ダウンロードします。
  </Step>
  <Step num="4" title="起動オプションの設定">
    Steamでゲームを右クリックし、「プロパティ」を選択して、起動時にクラッシュする場合は `-dx12` や `-windowed` などの起動オプションを追加します。
  </Step>
</Steps>

<Callout type="tip" title="シェーダーのプリキャッシュ">
最初の数レベルの読み込み中に発生するゲームプレイの初期のカクつきを抑えるために、Steamのシェーダープリキャッシュ機能を有効にしてください。
</Callout>

</Section>

<Section icon={Sparkles} title="パフォーマンスの最適化とグラフィック設定">

激しい戦闘シーンやパズルを解く場面で高いフレームレートを維持するには、グラフィック設定の微調整が不可欠です。Apple SiliconのGPUはテクスチャレンダリングを効率的に処理しますが、重いポストプロセス効果はパフォーマンスのボトルネックになる可能性があります。

ハードウェアの効率を最大化するには、ダイナミックシャドウやボリュメトリック効果などの負荷の高い設定を下げてください。解像度スケーリングや空間アップスケーリング技術を利用することでも、全体のフレームレートを大幅に向上させることができます。

| 設定 | 推奨値 | パフォーマンスへの影響 |
| :--- | :--- | :--- |
| **解像度** | 1920x1080（スケーリング） | 高いパフォーマンス向上効果 |
| **影の品質** | 中 | 中程度のパフォーマンス向上効果 |
| **アンチエイリアス** | Temporal AA または FXAA | パフォーマンスへの影響は低い |
| **テクスチャ品質** | 高（16GB RAMの場合） | 最新チップへの影響は低い |
| **垂直同期** | 有効 | 画面の引き裂き（ティアリング）を防止 |

パフォーマンスは、お使いのApple Siliconプロセッサの世代によって異なります。以下のベンチマークを確認し、お使いのシステム構成で現実的なパフォーマンスの期待値を設定してください。

| Appleチップ | 平均FPS | 快適さの評価 |
| :--- | :--- | :--- |
| **M1（ベースモデル・8GB）** | 35-45 FPS | プレイ可能 (★★★☆☆) |
| **M1 Pro / Max** | 60 FPS（安定） | 快適 (★★★★☆) |
| **M2 / M3（ベースモデル）** | 50-55 FPS | 非常に良好 (★★★★☆) |
| **M3 Pro / Max** | 90+ FPS | 極めて快適 (★★★★★) |

<Callout type="info" title="サーマルスロットリング">
MacBook Airモデルには冷却ファンが搭載されていません。30分ほどプレイした後にフレームドロップが発生する場合は、ノートPC用冷却パッドを使用するか、解像度を下げることを検討してください。
</Callout>

</Section>

<Section icon={Swords} title="コントローラーと入力の設定">

難易度の高い足場を渡ったり、戦闘を行ったりするには、正確な入力反応が必要です。互換レイヤーを介してキーボードとマウスでの操作もサポートされていますが、専用のゲームコントローラーを使用する方がはるかに直感的なレイアウトになります。

macOSは最新のコンソールコントローラーのBluetooth接続を標準でサポートしており、サードパーティのドライバーを必要とせずにSteamの入力設定システムと直接連携できます。

| アクション | Xbox コントローラー | PlayStation コントローラー |
| :--- | :--- | :--- |
| **移動** | 左スティック | 左スティック |
| **インタラクト** | A ボタン | × ボタン |
| **戦闘 / 攻撃** | X ボタン | □ ボタン |
| **回避** | B ボタン | ○ ボタン |
| **一時停止メニュー** | メニューボタン | オプションボタン |

ゲームを起動する前に、macOSの「システム設定」メニューからコントローラーが正しくペアリングされていることを確認してください。ペアリングされると、Steam入力が信号を変換し、仮想のWindows環境に適合させます。

<Callout type="success" title="コントローラーの互換性">
DualSenseとXboxワイヤレスコントローラーはどちらもmacOSでネイティブにサポートされており、CrossOverやWhisky内のSteamにシームレスに認識されます。
</Callout>

</Section>
