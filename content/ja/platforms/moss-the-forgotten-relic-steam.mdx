<Section icon={Target} title="デュアルコントロールシステムと初心者向けガイド">

この冒険の核心となるメカニクスは、2つの異なる存在によるユニークなパートナーシップです。プラットフォームアクションと近接戦闘を担当する勇敢な若いネズミの「クイル」と、プレイヤーが直接操作する巨大な魔法の存在「読む者（リーダー）」です。この世界で道を切り拓くには、これら両方のキャラクターを同時に連携させる必要があります。

クイルは、走る、ジャンプする、回避する、攻撃するといった標準的な方向入力で操作します。一方、「読む者」はマウスカーソルやコントローラーのアナログスティックを使用し、重いブロックを掴む、古代の仕掛けを回転させる、戦闘中にクイルを回復する、敵対するアーケイン（Arcane）の軍勢を拘束するなどの操作を行います。

| アクション | キーボード＆マウス | コントローラー (Xbox / PS) | ゲームプレイでの用途 |
| :--- | :--- | :--- | :--- |
| **クイルの移動** | WASD | 左アナログスティック | クイルの歩行、走行、クライミングを操作 |
| **クイルのジャンプ** | スペースキー | Aボタン / ×ボタン | 足場の移動や短い隙間を飛び越える際に使用 |
| **クイルの攻撃** | 左クリック | Xボタン / □ボタン | 基本的な剣の振り下ろしや近接コンボを実行 |
| **クイルの回避** | Shiftキー | Bボタン / ◯ボタン | 敵の攻撃を回避し、危険エリアから離脱する |
| **読む者のインタラクト** | 右クリック＋ドラッグ | 右トリガー＋アナログ | ブロックの移動、クイルの回復、アーケイン（敵）の操作 |

<Section icon={Sparkles} title="完全攻略とチャプター進行">

本作の物語は、命を吹き込まれた絵本のように構成されています。最も一貫した体験を得るためには、まず「Book I」をクリアし、「トワイライト・ガーデン」のポータルが出現したらその都度入り、その後「Book II」の拡張された世界へと直接移行することをおすすめします。

ゲームは、ジオラマ画面が切り替わるごとに自動でセーブされます。クイルが戦闘で倒れたり、足場を踏み外して落下したりしても、進行状況を失うことなく、現在の部屋の入り口から即座にリスポーンします。

| 書籍＆セクション | 収録チャプター | 主な目標 | ゲームプレイの核心 |
| :--- | :--- | :--- | :--- |
| **Book I: オープニング** | チャプター1〜2 | 「読む者」との絆を築き、故郷を旅立つ | 基本的な移動、シンプルなスイッチ、初期の戦闘 |
| **Book I: 遺跡** | チャプター3〜5 | マイル寺院を越え、ポータルを見つける | 感圧板、昇降プラットフォーム、パズル解決 |
| **トワイライト・ガーデン** | ポータル1〜3 | 任意の試練をクリアし、装備を獲得 | 高度なチャレンジルーム、隠された道の出現 |
| **Book I: フィナーレ** | チャプター6〜7 | サルフォグの領地を突破する | テンポの速いアスレチック、追跡シーケンス、ボス戦 |
| **Book II: 前半** | チャプター1〜4 | 仲間を集め、自然の力を手に入れる | ツタを成長させて登る、より広大なジオラマの管理 |
| **Book II: 後半** | チャプター5〜9 | 頂（ピナクル）を進み、タイランを倒す | チャージ武器パズル、鏡の反射ルート、最終ボス戦 |

<Callout type="warning" title="トワイライト・ガーデンへの進入">
Book Iのチャプター3、4、5に隠されているポータルには必ず入るようにしましょう。これらの試練を早期にクリアして強力な装備を手に入れておくと、後半のチャプターが大幅に楽になります。
</Callout>

「読む者」は戦闘中に直接クイルをサポートすることもできます。敵を掴むことで、クイルへの攻撃を防いだり、その敵を操って他の敵を攻撃させたりすることが可能です。
