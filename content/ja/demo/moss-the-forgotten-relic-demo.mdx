<Section icon={Swords} title="戦闘戦略：剣 vs ハンマー">

**Moss: 忘れられた秘宝 デモ**では、武器を通じて2つの異なるプレイスタイルが導入されます。最初は機動力に優れた標準的な剣から始まりますが、デモの真の主役は後半で見つかる「ハンマー」です。

| 特徴 | 秘宝の剣 | カリルのハンマー |
| :--- | :--- | :--- |
| **攻撃速度** | 非常に速い | 遅い / 重い |
| **ダメージタイプ** | 刺突 / 斬撃 | 打撃 / 範囲攻撃 |
| **特殊能力** | 高速ダッシュストライク | チャージ叩きつけトラップ |
| **パズルでの用途** | 蔓（つる）を切る | 重い感圧板の起動 |
| **有効な敵** | 小さな飛行昆虫 | 装甲を纏ったアーケインの番人 |

### ハンマーメカニクスの習得
ハンマーは単なる戦闘ツールではなく、パズルを解くためのデバイスでもあります。入手すると、床や重い仕掛けに特定の「ハンマーシンボル」があることに気づくでしょう。

<Steps>
  <Step num="1" title="ハンマーシンボルの特定">
    地面にある円形の光るプレートや、特定の柱を見つけます。
  </Step>
  <Step num="2" title="武器をチャージする">
    ハンマーがアーケインのエネルギーで光るまで、特殊攻撃ボタンを長押しします。
  </Step>
  <Step num="3" title="ターゲットを叩く">
    シンボルの上で攻撃を放ち、プラットフォームの上昇やゲートの開放などの持続的な効果を誘発させます。
  </Step>
  <Step num="4" title="トラップのタイミング">
    一部のスイッチは、叩いた後にチャージが消える前に素早くクイルを次のエリアへ移動させる必要があります。
  </Step>
</Steps>

<Callout type="tip" title="戦闘の相乗効果">
「読む者」で敵を掴んで固定し、クイルに切り替えてチャージされたハンマーの一撃を放つことで、周囲の敵に巨大な範囲ダメージを与えることができます。
</Callout>

</Section>

<Section icon={Sparkles} title="ウォークスルー：温室とその先">

デモの主な舞台は、野生植物と古代の遺跡に満ちた青々と茂る温室です。進行は一本道ですが、各部屋の垂直方向の探索を行うプレイヤーには報酬が用意されています。

### デモの主要なロケーション
1.  **玉座の間への入り口：** ここでアーガスおじさんに会います。「読む者」の掴みメカニクスを使用して、大きな石の扉を引いて開ける必要があります。
2.  **沈んだ庭園：** 初めてアーケインと遭遇する戦闘アリーナです。敵は3体1組で攻撃してくるため、ここでは回避ロールを頻繁に使用してください。
3.  **ハンマーの保管庫：** 温室の中心部の下にある隠された部屋。敵を感圧板の上に立たせている間にクイルがジャンプして渡る、マインドコントロールパズルを解く必要があります。
4.  **番人の見晴台：** 「シンダーナイト」と対峙する、デモの最終エリアです。

| 収集アイテムの種類 | 目的 | 場所のヒント |
| :--- | :--- | :--- |
| **アーケインの塵** | アップグレード用の通貨 | 敵のドロップや壊せる壺から入手 |
| **伝承のスクロール** | 物語の背景知識 | 通常、高い足場や滝の裏に隠されています |
| **体力のレリック** | 恒久的なHP増加 | 主要な戦闘の後に見つかります |

<Section icon={Shield} title="ボス戦：シンダーナイトの番人">

**Moss: 忘れられた秘宝 デモ**は、シンダーナイトとの劇的な対決で幕を閉じます。このボスは、移動と武器のタイミングに関して学んだすべてのスキルの確認テストとなります。

| フェーズ | ボスの攻撃 | 対抗策 |
| :--- | :--- | :--- |
| **フェーズ 1** | 頭上からの地面叩きつけ | 横にロールして手を攻撃する |
| **フェーズ 1** | なぎ払い攻撃 | 刃が通り過ぎる時にジャンプして避ける |
| **フェーズ 2** | アーケインの飛び道具 | 「読む者」を使って物体で飛び道具をブロックする |
| **フェーズ 2** | ハンマーの脆弱性 | ハンマーをチャージして、青く光る弱点を叩く |

この戦いでは、シンダーナイトが時折その巨大な手をプラットフォームに置きます。これがダメージを与えるチャンスです。ハンマーを装備している場合、拳にチャージ攻撃を当てると敵がスタンし、クイルが顔面にフルコンボを叩き込むことができます。

<Callout type="warning" title="環境の危険">
アリーナの床が時々赤く光ります。これは範囲攻撃（AoE）の予兆です。甚大なダメージを避けるために、クイルをプラットフォームの外縁へ移動させてください。
</Callout>

</Section>
