<Section icon={Shield} title="『Moss: 忘れられた秘宝』コープ：公式マルチプレイ対応状況">

多くのプレイヤーがこの待望の非VRコレクション版のリリースに沸く中、**『Moss: 忘れられた秘宝』のコープ（協力プレイ）**対応についての疑問が頻繁に寄せられています。公式には、Polyarc Gamesは本作を**シングルプレイ体験**として設計しています。このゲームは、プレイヤーが霊的な「読む者」となってネズミの主人公「クイル」を見守るというオリジナルのVRメカニクスを、フラットスクリーン（非VR）形式に再構築したものです。

このバージョンでは、2つの異なる操作系統を同時に担当することになります。オンラインのマッチングやローカルの画面分割モードはありませんが、プレイヤー自身が担う2つの役割のシナジー（相乗効果）によって、ゲームプレイに「協調的」な性質が生まれます。

| 機能 | 対応状況 | 詳細 |
| :--- | :--- | :--- |
| **オンラインコープ** | **非対応** | 公式のオンラインマルチプレイやマッチングはありません。 |
| **ローカルコープ** | **非対応** | ネイティブの画面分割や、2人目のローカルプレイヤーサポートはありません。 |
| **デュアルコントロール** | **対応** | 1人のプレイヤーがクイルと「読む者」の両方を操作します。 |
| **シェアプレイ** | **一部対応** | Steam Remote Play TogetherやPS5 Share Playを使用してストリーミング可能です。 |
| **クロスプレイ** | **対象外** | シングルプレイ設計のため、該当しません。 |

<Callout type="warning" title="開発者からの説明">
ゲームの基本ループは、「読む者」とクイルの間の深く、一対一の絆に依存しています。2人目のプレイヤーを導入するには、現在1人のプレイヤーが両方のキャラクターを管理することを前提にバランス調整されている環境パズルを、完全に再設計する必要があります。
</Callout>

<Section icon={Target} title="擬似的に協力プレイ（コープ）を楽しむ方法">

**『Moss: 忘れられた秘宝』のコープ**は内蔵機能ではありませんが、本作は「お茶の間コープ」や共同プレイに非常に適しています。ゲームがまるで生きている絵本のように進行するため、多くのプレイヤーが、認知的な負担を分担してパートナーと一緒にプレイすることにやりがいを感じています。

1人のプレイヤーがクイルの物理的な操作に集中し、もう1人が「参謀」として**遺物の粉（Relic Dust）**などの隠された収集品を見つけたり、パズルの解き方をアドバイスしたりすることができます。

**ビデオハイライト:**
- *注意: 現在、このガイド用の公式ゲームプレイビデオはデータベースに登録されていません。ローンチトレーラーについては、Polyarcの公式YouTubeチャンネルを参照してください。*

<Steps>
  <Step num="1" title="「助手席」のパイロット">
    1人がコントローラーを操作し、もう1人が画面を観察して、操作しているプレイヤーが見落としがちな隠し通路、敵の出現位置、パズルのスイッチなどを発見します。
  </Step>
  <Step num="2" title="パッド回し戦略">
    チャプターごとに役割を交代します。戦闘の激しい**『Book II』**のセクションを1人が担当し、もう1人が**『黄昏の庭園』**の複雑なパズルに挑むといった分担が可能です。
  </Step>
  <Step num="3" title="配信によるコラボレーション">
    DiscordやPlayStationのシェアプレイを使用して、フレンドにリアルタイムで画面を見てもらいながらパズルを解くことで、ゲームをソーシャルな謎解きイベントに変えることができます。
  </Step>
</Steps>

<Callout type="tip" title="コープのプロのコツ">
PCでプレイしている場合は、**Universal Split Screen**などのソフトウェアを使用するか、キー割り当てを変更して、1人がマウス（読む者）を操作し、もう1人がWASDキー（クイル）を操作するように設定することも可能です（ただし、息を合わせるのがかなり難しくなります！）。
</Callout>

</Section>

<Section icon={Map} title="デュアルキャラクターシステムのマスター">

2人目のプレイヤーなしで攻略するには、クイルと「読む者」の相互作用をマスターする必要があります。これこそが、**『Moss: 忘れられた秘宝』におけるコープ**の代替要素の核心です。ゲームは「スマートフォローカメラ」を採用しており、両方のキャラクターが常に視野に入るようになっていますが、プレイヤーは片手でクイルを動かし、もう片方の手で世界を操作することを学ぶ必要があります。

| 役割 | 主な入力 | よくあるインタラクション |
| :--- | :--- | :--- |
| **クイル** | 左スティック / ボタン | ジャンプ、攻撃、回避、登る、ダッシュ |
| **読む者** | 右スティック / トリガー | ブロックを掴む、歯車を回す、回復、敵の減速 |
| **組み合わせ** | 両方同時に操作 | クイルが上に乗っている間にブロックを動かす |

<Callout type="success" title="シナジーメカニクス">
**『Book II』**では、この相乗効果がさらに拡張されます。「読む者」の力を使ってクイルの武器（チャクラムやハンマーなど）をチャージすることができ、ソロプレイであっても「チームアタック」のような感覚を味わうことができます。
</Callout>

**重要なデュアルアクション戦略:**
- **戦闘中の回復**: クイルがスクリーチャー（Screecher）の攻撃を回避している間、「読む者」のカーソルをクイルに合わせてインタラクションボタンを長押しし、体力を回復させます。
- **敵の操作**: カブトムシのようなアーケインの敵を「読む者」で掴み、感圧板の上に移動させることで、クイルが時間制限のあるゲートを通り抜けられるようにします。
- **環境の架け橋**: クイルが隙間を飛び越える間、「読む者」で回転する足場を固定します。手を離すと足場がリセットされ、クイルが落下してしまう可能性があります。

</Section>

<Section icon={Swords} title="2人で挑む戦闘とボス戦">

『Moss: 忘れられた秘宝』の戦闘は、**『Book II』**の後半ステージで大幅に複雑化します。**ウォードン（The Warden）**や**タイラン（Tylan）**などのボス戦では、攻撃と世界の操作を素早く切り替える必要があります。近くに友人がいる状態でプレイしているなら、こここそが「コープ」の真骨頂となります。

| ボス / エンカウント | 「読む者」の優先事項 | クイルの優先事項 |
| :--- | :--- | :--- |
| **サーフォグ（大蛇）** | 足場の移動 / ゲート操作 | 素早い登攀と回避 |
| **ウォードン** | ハンマートラップの起動 | 露出した弱点への攻撃 |
| **タイラン（フクロウ）** | 鐘の操作 | ソードダッシュとチャクラムの投擲 |
| **アーケインの群れ** | 遠距離攻撃を行う敵の気絶 | 近接の脅威の排除 |

<Callout type="info" title="アクセシビリティに関する注意点">
同時に2つのタスクをこなすのが難しすぎる場合、ゲームには**「戦闘をスキップ（Skip Combat）」**機能が用意されています。「ソロコープ」での戦闘が厳しすぎると感じた場合は、ストーリーとパズルだけに集中することができます。
</Callout>

**「チーム」向けの戦闘のコツ:**
- **フォーカスファイア（集中攻撃）**: 「観測者（2人目）」は、どの敵が飛び道具を撃とうとしているかを声に出して伝えます。
- **リソース管理**: 操作しているプレイヤーは目の前の戦闘に集中しがちなので、遺物の粉のメーターやクイルの体力バーを代わりに監視してあげましょう。
- **パズル戦闘**: パズルを解くために生かしておかなければならない敵もいます。クイルを操作しているプレイヤーが、感圧板に必要なアーケインを誤って倒してしまわないように注意してください。

</Section>
