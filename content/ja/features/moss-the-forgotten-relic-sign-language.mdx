<Section icon={MessageSquare} title="一般的な手話とその意味">

**『Moss: 忘れられた秘宝』の手話**を理解することは、環境パズルを解き、物語をより深く味わうための鍵となります。ゲームをクリアするためにASLに精通している必要はありませんが、いくつかの主要なサインを覚えるだけで体験が大幅に向上します。クイルのアニメーションは、指の一本一本の動きが正確で認識しやすいものになるよう、ろう者コミュニティの協力を得て手作業で制作されました。

本作では、手話は主に3つのタイプに分類されます：**社交的手話**、**指示的手話**、**感情的な手話**です。社交的なものはプレイヤーとの交流に、指示的なものは「泥の寺院」や「見捨てられた街」にある複雑なパズルを解くのに役立ちます。

| 手話のカテゴリー | 一般的なジェスチャー | 文脈における意味 | プレイヤーの反応 |
| :--- | :--- | :--- | :--- |
| **社交的** | **胸に平手を当てる** | **ありがとう** | クイルの頭を撫でる |
| **社交的** | **手を振る** | **こんにちは / やあ** | プレイヤーのオーブを動かす |
| **指示的** | **対象物を指差す** | **見て / これを動かして** | その対象物を操作する |
| **指示的** | **耳の近くに手を置く** | **聞いて / 危険** | 戦闘に備える |
| **感情的** | **両手を顔に当てる** | **怖い** | クイルを回復・安心させる |
| **感情的** | **心臓の上で拳を握る** | **勇気 / 準備完了** | 次のエリアへ進む |

<Callout type="success" title="インタラクションのヒント">
クイルを回復させたり、段差を越えるのを手伝った後に彼女が「ありがとう」の手話をしたとき、プレイヤーのインタラクションボタンを使って「ハイタッチ」をしたり、彼女の頭を撫でたりして応えることができます。これにより絆が深まり、時にはユニークなアニメーションがトリガーされることもあります。
</Callout>

</Section>

<Section icon={Target} title="クイルのヒントの読み解き方">

**『Moss: 忘れられた秘宝』の手話**の最も役立つ側面の一つは、組み込まれたヒントシステムです。パズルルームでプレイヤーが1分以上操作を行わない場合、クイルは積極的に助けようとします。彼女はただ立っているだけでなく、周囲をスキャンし、ASLを使ってあなたのカーソルを導きます。

<Steps>
  <Step num="1" title="クイルの構えを観察する">
    クイルが後ろ足で立ち上がり、カメラの方を向いたら、それは何かを伝えようとしている合図です。オーブを動かすのを止めて、彼女のアニメーションが終わるのを待ちましょう。
  </Step>
  <Step num="2" title="ターゲットを特定する">
    クイルは通常、注意が必要な特定のブロック、スイッチ、または感圧板を指差します。『忘れられた秘宝』では、彼女が指差したオブジェクトがかすかに緑色に光ることがよくあります。
  </Step>
  <Step num="3" title="アクションを解釈する">
    彼女が押すような動作を真似ているなら、そのオブジェクトを動かしてほしいということです。引くような動作なら、レバーや機構を作動させるよう求めています。
  </Step>
  <Step num="4" title="解決策を実行する">
    彼女の導きに従い、ハイライトされたオブジェクトをインタラクションボタンで掴んでください。正しいアクションが行われると、クイルはよく「親指を立てる（グッジョブ）」や「ありがとう」の手話をします。
  </Step>
</Steps>

<Callout type="warning" title="パズルの難易度設定">
クイルのヒントは助けになるように設計されていますが、押し付けがましいものではありません。パズルを完全に自力で解きたい場合は、アクセシビリティメニューで「積極的なヒント」を無効にすることができます（ただし、物語上必要な場面では引き続き手話を使用します）。
</Callout>

</Section>

<Section icon={Sparkles} title="非VR版における手話">

VRから標準的なスクリーンへの移行は、Polyarc Gamesにとって独自の挑戦でした。それは、**『Moss: 忘れられた秘宝』の手話**の視認性をいかに維持するかという点です。オリジナルのVR版では、プレイヤーが身を乗り出してクイルの小さな手を確認できましたが、2026年の非VR版では、この機能を維持するためにいくつかの技術的アップグレードが実施されました。

「スマートフォローカメラ」は最も重要な追加要素です。このシステムは、クイルのアクションに基づいて視野角とズームレベルを自動的に調整します。彼女がASLのシークエンスを開始すると、カメラが微妙にショットを絞り、彼女のジェスチャーが画面中央にはっきりと映るようにします。さらに、クイルの毛並みや足の質感が高精細になったことで、Nintendo Switch 2やSteam Deckのような携帯デバイスでも手話がこれまで以上に鮮明に見えるようになりました。

| 機能 | VR版の詳細 | 非VR『忘れられた秘宝』の詳細 |
| :--- | :--- | :--- |
| **カメラ制御** | 手動のヘッドトラッキング | **自動スマートフォロー** |
| **視認性** | プレイヤーとの距離に依存 | **手話時のダイナミックズーム** |
| **ハンドモデル** | 標準解像度 | **強化された4Kテクスチャ** |
| **フレームレート** | 90 FPS (ヘッドセット) | **目標 60/120 FPS (平面)** |
| **ヒントの明快さ** | 立体音響の合図 | **視覚的な手話の強調** |
