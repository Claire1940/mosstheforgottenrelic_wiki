<Section icon={Map} title="マルチプレイヤーの状況とデュアルコントロールの仕組み">

この新しい非VR版をプレイし始めた多くのプレイヤーが、**Moss: 忘れられた秘宝 マルチプレイヤー**機能に関する情報を探しています。はっきりさせておくと、**『Moss: 忘れられた秘宝』には、伝統的なオンラインまたはローカルの協力マルチプレイヤーは搭載されていません。** このゲームは、ネズミのヒーロー「クイル」と、あなたである「読む者」との間のユニークな感情的・機械的な絆を強調するシングルプレイヤーの旅としてゼロから設計されています。

マッチメイキングのロビーや2人目のプレイヤーを招待するシステムはありませんが、ゲームプレイはしばしば「自分自身との協力プレイ」のように感じられます。あなたは常に2つの異なる入力を管理する責任があります。クイルは物理的な移動や剣術を担当し、読む者は周囲の世界を操作して道を切り開き、サポートを提供します。

| 役割 | 主な責任 | 操作タイプ |
| :--- | :--- | :--- |
| **クイル** | 走る、跳ぶ、登る、回避、近接戦闘 | キャラクターの直接移動 |
| **読む者** | ブロックの移動、クイルの回復、敵のスタン、スイッチの起動 | 環境への干渉 |
| **絆** | 体力の共有、パズルの連携、感情的なストーリーテリング | 統一された進行 |

<Callout type="tip" title="絆をマスターする">
このゲームを、左手でプラットフォームアクションを、右手で環境操作を行うシングルプレイヤーの「協力プレイ」と考えてください。成功の鍵は、一方だけに集中するのではなく、両方のキャラクターを同期させて動かすことにあります。
</Callout>

**2026年7月16日**にリリースされた非VR版は、オリジナルのVRにおける「手を伸ばす」メカニクスを、コントローラーやマウス＆キーボードの設定に適応させています。これにより、もともとモーションコントロール用に設計された複雑なパズルも、標準的なディスプレイで満足感のある直感的な操作が可能になっています。

</Section>

<Section icon={Target} title="初心者ガイド：デュアルコントロールをマスターする">

**Moss: 忘れられた秘宝 マルチプレイヤー**という選択肢がない以上、クイルと読む者の両方を効果的に操る方法を学ぶ必要があります。以下のステップは、第1戦記の序盤を攻略し、2つの役割の間のリズムを確立するのに役立ちます。

<Steps>
  <Step num="1" title="絆を初期化する">
    まずは**第1戦記**から始めて、基本的なパズルのルールを学びましょう。最初にクイルに出会ったら、読む者の光を使って彼女に干渉します。これにより、彼女を回復させたり、彼女のすぐ近くにあるオブジェクトを操作したりできるリンクが確立されます。
  </Step>
  <Step num="2" title="インタラクションマークを見つける">
    棚や壁にある白いひっかき傷を探してください。これらはクイルが登れる道を示しています。同様に、かすかに緑や青に光るオブジェクトは、読む者が操作できるものです。クイルを動かす前に、どのブロックを引きずったり回転させたりできるかを確認しましょう。
  </Step>
  <Step num="3" title="戦闘と回復を連携させる">
    戦闘では、クイルを使って回避し、攻撃します。彼女の体力（光るエンブレムで表示）が低下したら、読む者で彼女を「掴み」、インタラクションボタンを押し続けて回復させます。また、敵を掴んでスタンさせることで、クイルが攻撃する隙を作ることもできます。
  </Step>
  <Step num="4" title="環境パズルを解く">
    ほとんどの部屋では、読む者で仕掛けを固定している間にクイルが開いたゲートを通過する必要があります。手を離すとプラットフォームがリセットされる場合は、クイルが次の安定した足場に安全に渡るまで、読む者のカーソルをオブジェクトに合わせたままにしてください。
  </Step>
</Steps>

<Callout type="success" title="戦闘のアクセシビリティ">
激しい戦闘中に両方のキャラクターを管理するのが難しすぎると感じた場合、ゲームには**戦闘をスキップ**するプロンプトが含まれています。これにより、ペナルティなしで戦闘を回避し、ストーリーやパズルを続けることができます。
</Callout>

</Section>

Mossにおける戦闘は、クイルと同じくらい読む者の役割も重要です。読む者を使って戦場を操作することは、特に第2戦記の後半の章で生き残るために不可欠です。

| 敵のタイプ | 読む者の戦略 | クイルの戦略 |
| :--- | :--- | :--- |
| **スクリーチャー** | スタンさせて突進を止める | 素早い3連コンボ |
| **ティッカー** | 他の敵の方へ引きずる | 爆発範囲から回避する |
| **スコーチャー** | 火の弾をブロックする | 側面や背面から攻撃する |
| **フォージ・ユニット** | 盾をこじ開けたままにする | ハンマーで大ダメージを与える |

<Callout type="info" title="自然との調和">
第2戦記では、**自然との調和**をアンロックします。この読む者の力により、ツタを成長させたり橋を架けたりして、以前訪れたエリアに新しい道を作ることができます。これは隠された収集アイテムを見つけるための鍵となります。
</Callout>
