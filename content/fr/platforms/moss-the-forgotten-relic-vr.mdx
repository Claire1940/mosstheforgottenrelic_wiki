<Section icon={Target} title="Guide du Débutant : Maîtriser le système de double contrôle">

Le cœur de **Moss : La relique oubliée VR** réside dans le partenariat entre deux personnages distincts. Quill est la courageuse souris protagoniste que vous guidez à travers le monde, tandis que le Lecteur (vous) est une présence divine capable de manipuler l'environnement. Le succès dépend de votre capacité à coordonner ces deux rôles simultanément.

<Callout type="tip" title="Conseil d'interaction">
Le curseur du Lecteur peut faire plus que déplacer des blocs. Vous pouvez intervenir dans le monde pour soigner Quill en maintenant le bouton d'interaction sur elle, ou même étourdir les ennemis pour lui créer une ouverture.
</Callout>

<Steps>
  <Step num="1" title="Initialiser le lien">
    Commencez par le Livre I. Bien que le Livre II soit accessible immédiatement, le premier livre établit le langage mécanique du jeu et introduit l'histoire dans l'ordre prévu.
  </Step>
  <Step num="2" title="Apprendre les commandes séparées">
    Quill gère les phases de plateforme (courir, sauter, grimper) et le combat au corps à corps. Le Lecteur utilise un curseur pour saisir des blocs, faire pivoter des mécanismes et tirer des grilles lourdes. Entraînez-vous à déplacer Quill tout en faisant pivoter une plateforme pour maîtriser le flux.
  </Step>
  <Step num="3" title="Scanner les indices visuels">
    Cherchez les marques de griffures blanches sur les rebords et les murs. Ce sont des indicateurs universels montrant que Quill peut grimper sur cette surface. Si un rebord est trop haut, cherchez un bloc à proximité que le Lecteur peut mettre en position.
  </Step>
  <Step num="4" title="Préparer l'itinéraire">
    Avant de déplacer Quill dans une zone dangereuse, utilisez le Lecteur pour aligner les plateformes et activer les interrupteurs. Construire le chemin à l'avance évite les réinitialisations inutiles et maintient un bon rythme.
  </Step>
</Steps>

<Callout type="warning" title="Conscience du combat">
Ne battez pas tous les ennemis immédiatement. De nombreuses énigmes nécessitent d'utiliser le Lecteur pour maintenir un ennemi Arcanique (ressemblant à un scarabée) sur une plaque de pression afin de garder une porte ouverte pour Quill.
</Callout>

</Section>

<Section icon={Sparkles} title="Solutions aux énigmes et mécaniques environnementales">

Les énigmes dans **Moss : La relique oubliée VR** fonctionnent souvent comme des "dioramas vivants". Si vous êtes bloqué, travaillez à rebours depuis la sortie. Identifiez ce qui bloque le chemin et déterminez quelle capacité du Lecteur peut l'éliminer.

| Type d'énigme | Mécanique requise | Solution courante |
| :--- | :--- | :--- |
| **Corniche haute** | Mouvement du Lecteur | Faire glisser un bloc ou une statue sous le rebord griffé. |
| **Pont rotatif** | Contrôle du Lecteur | Maintenir le mécanisme en place pendant que Quill traverse. |
| **Plaque de pression** | Manipulation Arcanique | Étourdir un ennemi et le traîner sur l'interrupteur. |
| **Plateforme de verre** | Pouvoir de révélation | Utiliser la capacité de révélation du Lecteur pour afficher les chemins cachés. |
| **Pont de lianes** | Harmonisation Nature | Faire pousser du lierre pour relier deux plateformes déconnectées. |
| **Grille lourde** | Marteau de Cariel | Briser le verrou blindé avec une frappe de marteau chargée. |

<Callout type="tip" title="L'observation est la clé">
Si une plateforme se réinitialise automatiquement lorsque vous la lâchez, cela signifie que le Lecteur doit continuer à la maintenir pendant que vous guidez Quill de l'autre main.
</Callout>

</Section>

Les rencontres de boss dans **Moss : La relique oubliée VR** sont des énigmes à étapes multiples. Vous ne pouvez pas simplement "forcer" ces combats ; vous devez utiliser l'environnement et les pouvoirs du Lecteur pour créer des ouvertures pour Quill.

| Boss | Stratégie Phase 1 | Stratégie Phase 2 | Rôle du Lecteur |
| :--- | :--- | :--- | :--- |
| **Sarffog** | Esquiver le souffle de feu et grimper. | Naviguer sur des corniches étroites s'effondrant. | Déplacer les plateformes rapidement. |
| **Le Gardien** | Éviter les zones d'écrasement orange. | Charger les Pièges à Marteau sur les points faibles. | Retenir le bras du boss. |
| **Tylan** | Utiliser les Pièges à Marteau sur les frappes au sol. | Poursuite sur des plateformes mobiles. | Charger le Chakram pour la cloche. |

<Callout type="warning" title="Note sur l'accessibilité">
Si un combat de boss est trop difficile, vous pouvez utiliser l'option **Passer le combat** qui apparaît dans le menu. Cela vous permet de progresser dans l'histoire tout en conservant les éléments d'énigme.
</Callout>
