<Section icon={Shield} title="Configuration requise et performances">

Afin de garantir une expérience de jeu fluide, l'équipe de développement a optimisé le moteur de jeu pour le matériel moderne. Les joueurs PC devront s'assurer que leur système répond à la configuration minimale requise pour faire tourner le jeu à un taux de rafraîchissement stable. Le jeu utilise des effets d'éclairage et de particules avancés, qui bénéficient grandement d'un stockage SSD.

| Composant | Configuration minimale | Configuration recommandée |
| Système d'exploitation | Windows 10 (64 bits) | Windows 11 (64 bits) |
| Processeur | Intel Core i5-8400 | Intel Core i7-10700K |
| Mémoire | 8 Go de RAM | 16 Go de RAM |
| Carte graphique | NVIDIA GTX 1060 | NVIDIA RTX 3060 |
| Stockage | SSD de 50 Go | SSD de 50 Go |

Les versions console proposeront des modes dédiés aux performances et à la qualité. Le mode performance vise un taux stable de 60 images par seconde à une résolution dynamique, tandis que le mode qualité privilégie une résolution 4K native avec des fonctionnalités graphiques améliorées telles que les reflets par lancer de rayons (ray-tracing) et des ombres de meilleure qualité.
